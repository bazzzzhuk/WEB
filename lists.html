Списки

Список определений

Позволяет создавать списки Терминов с их разъяснением.

<dl> Definitions list;
<dt> Description term;
<dd> Description details;
<dt> u <dd> всегда используются парой.
Нумерованный список
Позволяет нумеровать элементы текста. Это нужно для перечисления в строгом порядке.
Маркированный список
Позволяет нумеровать элементы текста. Это нужно для перечисления в любом порядке.
Список определений
Позволяет перечислять понятия и давать им определения.

Нумерованный список

<ol> Order list;
<li> list item;
  1. List item 1
  2. List item 2
  3. List item 3
  4. List item 4
  5. List item 5

При необходимости список можно прервать и продолжить позже.

  1. List item 6
  2. List item 7
  3. List item 8
  4. List item 9
  5. List item 10

Маркированный список

<ul> UnOrder list;
<li> list item;
  • List item 1
  • List item 2
  • List item 3
  • List item 4
  • List item 5
